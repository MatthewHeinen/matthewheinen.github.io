Final Projects

Concurrent Binary Search Tree:

A project using Java Concurrency that created a BST which allowed for concurrent insert functions to occur. Concurrent insertion was created recursively by ensuring that we only were accessing two nodes at a time by using semaphores while allowing the rest of the tree to be open and allow other insertions to take place. To ensure there are no race conditions with other features of the BST, monitors were implemented to ensure that a select amount of inserts could finish before any other feature could commence.

Big Int:

A project written in C and C++ that allowed any integer greater than Integer Max to be displayed. This was achieved by ensuring that once a number reached Integer Max, this would carry and be represented as a one in the next column. New columns can keep being created depending on the size of the integer.

Puzzle Pieces:

This was a final project for my Intro to Computer Science course. This was a Python project showing how many combinations of puzzle pieces would create complete puzzles given user created pieces. An interface asked users what size puzzle square they would like to complete. A piece was determined with a list of arrays and having the user input 1’s or 0’s with a 1 meaning that square is being used as a square for that puzzle block. Using Numpy the pieces the user inputted into the program were rotated and shifted in every possible way and anytime a full puzzle was created we added it to a counter

Airline Reservation System:

I created the algorithm for the priority queue which first divided users into their status with our mock airline and then further divided these users on how long they have been waiting in the queue
