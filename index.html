Concurrent Binary Search Tree:

A project using Java Concurrency that created a BST which allowed for concurrent insert functions to occur. Concurrent insertion was created recursively by ensuring that we only were accessing two nodes at a time by using semaphores while allowing the rest of the tree to be open and allow other insertions to take place. To ensure there are no race conditions with other features of the BST, monitors were implemented to ensure that a select amount of inserts could finish before any other feature could commence.

A project written in C and C++ that allowed any integer greater than Integer Max to be displayed.
